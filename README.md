## 실행방법 

```sh
$ cd <프로젝트 디렉터리>
$ npm install
$ npm start
```

## 주요기능

<ul>
  <li> React Hook 함수형 컴포넌트 </li> 
  <li> React Material-UI Framework  </li>
  <li> Redux Toolkit 상태관리 </li>  
  <li> react-router-dom v6 라우팅 </li>
  <li> react-beautiful-dnd drag and drop 기능 </li> 
</ul>
